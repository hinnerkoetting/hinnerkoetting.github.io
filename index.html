Bumping Bunnies

Bumping bunnies is a remake of Jump'n Bump.
Play against friends on a single PC, via WLAN or Bluetooth.

Download

Android » PC and MAC »

Instructions for PC and MAC

Download bumping-bunnies.jar
Java 8 is required.

  1. Install Java 8 if necessary.
  2. Double click bumping-bunnies.jar
    Or start it via command line: java -jar bumping-bunnies.jar

Open source

Bumping Bunnies is open source. You can view the source on Github.
